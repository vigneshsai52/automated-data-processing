Automated Data Processing & Reporting System

Overview
This is a Python-based automation project that processes CSV data, cleans duplicates and missing values, and generates summary reports automatically.

Features

Loads CSV datasets using pandas

Removes duplicate records

Handles missing values

Generates summary statistics

Exports cleaned data and reports

Project Structure
automated-data-processing
data → input dataset
src → Python source code
output → generated output files (ignored in Git)
README.md
requirements.txt
.gitignore

How to Run

Install required package
pip install -r requirements.txt

Go to source folder
cd src

Run the program
python main.py

Output

Cleaned dataset saved as CSV file

Summary report saved as text file

Use Case
This project automates manual data cleaning and reporting tasks and demonstrates basic Python automation and data processing skills.
